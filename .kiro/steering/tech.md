# Technology Stack

## Runtime & Build System

- **Runtime**: Bun (JavaScript runtime with built-in SQLite)
- **Language**: TypeScript with strict mode enabled
- **Build Tool**: TSOA for route and OpenAPI generation

## Core Dependencies

- **Framework**: Express 5.x
- **API Documentation**: TSOA 6.x (TypeScript OpenAPI)
- **Database**: SQLite via `bun:sqlite` with WAL mode
- **Swagger UI**: swagger-ui-express for interactive docs
- **Validation**: Built-in TSOA validation

## TypeScript Configuration

- Target: ES2020
- Module: ESNext with bundler resolution
- Decorators: Experimental decorators enabled
- Strict mode: Enabled

## Common Commands

```bash
# Install dependencies
bun install

# Development (with hot reload)
bun run dev

# Production start
bun run start

# Generate routes and OpenAPI spec
bun run build
# or manually:
bunx tsoa spec    # Generate OpenAPI spec
bunx tsoa routes  # Generate Express routes
```

## TSOA Configuration

- Entry file: `src/server.ts`
- Controllers: `src/controllers/**/*.ts`
- Generated routes: `src/routes.ts` (auto-generated, do not edit)
- OpenAPI spec: `public/openapi.json` (auto-generated)
- OpenAPI version: 3.0

## Important Notes

- `src/routes.ts` is auto-generated by TSOA - never edit manually
- `public/openapi.json` is auto-generated - regenerate with `bunx tsoa spec`
- After adding/modifying controllers, run `bun run build` to regenerate routes
