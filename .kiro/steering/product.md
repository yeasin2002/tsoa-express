# Product Overview

A full-featured Todo API demonstrating automatic OpenAPI generation from TypeScript decorators using TSOA.

## Core Features

- User management (CRUD operations)
- Todo management (CRUD operations with filtering)
- Auto-generated OpenAPI/Swagger documentation
- SQLite database with WAL mode
- Type-safe API with runtime validation
- Query parameters for filtering and pagination

## Key Endpoints

- `/users` - User CRUD operations
- `/todos` - Todo CRUD operations with user association
- `/docs` - Interactive Swagger UI
- `/public/openapi.json` - OpenAPI 3.0 specification

## Development Server

- Default port: 9000
- Base URL: `http://localhost:9000`
- Hot reload enabled in dev mode
